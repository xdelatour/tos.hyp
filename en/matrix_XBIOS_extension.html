4.14 MATRIX XBIOS extension

The driver for the MATRIX graphic card installs some XBIOS functions.

MatGraph - Cxx

  • Unknown = 0x7D01 = 32001
  • CHECKinst = 0x7D02 = 32002
  • INSTokay = 0x43784f6b ; 'CxOk'

XBRA "M_IN"
